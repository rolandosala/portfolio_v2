Registrar Information System

Overview

This project is made to aid some process, tracking of requests and analytics of data from the place that I've worked on. I've seen that it needs an internal application that can help in recording and tracking of request.

Features

  • Record keeping and tracking of requests
  • Certificate and Transcript generation via PDF
  • Report generation
  • Analytics

Tech Stack

  • Frontend: Vuetify
  • Backend: Node.js, Express.js and MySQL
  • Others: Axios
